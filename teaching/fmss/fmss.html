Further Mathematics, Systems and Signals

This course consists of 11 sessions aimed at students in electrical and electronic engineering, introducing topics in Laplace transforms, Fourier series, and matrix algebra.


The full set of notes can be downloaded here, and the complete set of materials for the course (including lecture slides) can be downloaded here.


Sessions:
  1. Introduction to Laplace transforms: lecture slides, summary lecture handout.
  2. Laplace transforms - the delay theorem: lecture slides, summary lecture handout.
  3. Using Laplace transforms to solve ODEs: lecture slides, summary lecture handout.
  4. Laplace transforms - transfer functions: lecture slides, summary lecture handout.
  5. Bode plots, and introduction to Fourier series: lecture slides, summary lecture handout.
  6. Determing the complex form of Fourier series using Laplace transforms: lecture slides, summary lecture handout.
  7. Circuit analysis using Fourier series: lecture slides, summary lecture handout.
  8. Introduction to matrices: lecture slides, summary lecture handout.
  9. Calculating eigenvalues and eigenvectors: lecture slides, summary lecture handout, tutorial questions, and tutorial solutions.
  10. State variable description of ODE systems: lecture slides, summary lecture handout.
  11. Solving ODE systems using matrices: lecture slides, summary lecture handout.
